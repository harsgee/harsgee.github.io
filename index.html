#简介

HTML 是用来描述网页的一种语言。它通过使用标记标签来标记内容，使得网页能被浏览器根据标记标签的特性来显示。

HTML 通过标签标记页面内容，然后通过对这些标签进行设置属性和值使得内容能显示不同的效果。

一个HTML网页就是一个以html为文件后缀名的文档，你可以把它跟word文档相比。

word文档中，你可以设置文字大小、文字颜色、文字粗细、文字斜体、行间距等各种样式。
HTML 中，你也可以设置这些内容的属性，但是它不像word文档中可以有界面让你设置，而是
需要你通过一些代码来进行设置。

一个网页通常由三种内容组成，html + CSS + Javascript；其中，html就相当于word文档之中的内容，文字、图片等；
CSS就是你对HTML文档的内容进行设置的样式；
javascript就是实现一些逻辑功能，比如说在网页中，你用鼠标点击一个地方然后出现什么效果。
